# 📊 Excel Analysis

This folder contains the Microsoft Excel workbooks used for data exploration, cleaning, validation, and preliminary analysis for the **AI Pulse 2030 – Global AI Ecosystem Analysis** project.

## Contents

- Raw data exploration
- Data cleaning and preprocessing
- Pivot Tables and Pivot Charts
- KPI calculations
- Lookup functions and formulas
- Data validation
- Exploratory Data Analysis (EDA)

## Excel Features Used

- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Power Query
- XLOOKUP / VLOOKUP
- INDEX & MATCH
- IF, SUMIFS, COUNTIFS
- Text & Date Functions
- Data Validation
- Sorting & Filtering

## Purpose

The Excel analysis serves as the foundation for data preparation, quality checks, and business insights before performing SQL analysis and developing the Power BI dashboard.

## Output

The cleaned and validated datasets generated from this analysis are stored in the **Dataset/02_Clean** folder and are used for SQL queries and Power BI reporting.
