# 📊 Power BI

This folder contains the Power BI files, dashboards, and supporting assets for the **AI Pulse 2030 – Global AI Ecosystem Analysis** project.

## Contents

- Power BI report (.pbix)
- Dashboard PDF
- Dashboard screenshots
- Custom background images
- Icons and visual assets

## Dashboard Features

- Executive KPI Dashboard
- Interactive Filters & Slicers
- Drill-Down Analysis
- Trend & Time-Series Analysis
- Geographic Visualizations
- Funding & Investment Analysis
- Company & Industry Insights

## Power BI Skills Demonstrated

- Data Modeling
- Power Query (ETL)
- DAX Measures & Calculated Columns
- KPI Cards
- Bookmarks & Navigation
- Custom Tooltips
- Interactive Visualizations
- Performance Optimization

## Purpose

The Power BI dashboard transforms the cleaned and analyzed data into interactive visual reports, enabling stakeholders to explore AI ecosystem trends, investments, and business insights.

## Output

The dashboards in this folder present the final analytical insights derived from the Excel analysis, SQL queries, and cleaned datasets.
